# Accident Severity classification



## Requirements

This project requires the following dependencies:

- pandas
- matplotlib
- seaborn
- regex
- fasttext
- scikit-learn
- xgboost

These can be installed from the `requirements.txt` file.
using 

`pip install -r requirements.txt` command
